스마트폰 기초

전화, 문자, 설정, 필수 앱 다운로드 등 스마트폰의 가장 기본적인 사용법을 배워봅니다.

1. 전화 걸고 받기

스마트폰의 가장 기본적인 기능, 전화 걸고 받는 방법을 알아봅니다.

전화 걸기

  1. 전화 앱 열기: 홈 화면에서 전화 아이콘을 찾아서 누릅니다.
  2. 번호 입력: 숫자 키패드가 나타나면 전화할 번호를 누릅니다.
  3. 통화 버튼 누르기: 초록색 통화 버튼을 누르면 전화가 걸립니다.

전화 받기

  1. 초록색 버튼 밀기: 전화가 오면 화면에 초록색 받기 버튼이 나타납니다. 이 버튼을 오른쪽으로 밀거나 누릅니다.
  2. 빨간색 버튼 밀기: 전화를 받기 싫을 때는 빨간색 거절 버튼을 왼쪽으로 밀거나 누릅니다.

연락처 저장하기

  1. 새 연락처 추가: 전화 앱에서 연락처 탭으로 이동 후 + 또는 새 연락처 버튼을 누릅니다.
  2. 정보 입력: 이름, 전화번호 등을 입력하고 저장을 누릅니다.

2. 문자 메시지 보내기

가족이나 친구에게 간단한 안부나 정보를 문자로 주고받는 방법을 배워봅니다.

문자 앱 열기

  1. 메시지 앱 찾기: 홈 화면에서 메시지 또는 문자 아이콘을 찾아서 누릅니다.
  2. 새 메시지 작성: 화면 오른쪽 아래에 있는 + 또는 새 메시지 버튼을 누릅니다.

메시지 작성 및 전송

  1. 받는 사람 입력: '받는 사람' 칸에 상대방의 전화번호를 직접 입력하거나, 연락처 버튼을 눌러 저장된 연락처에서 선택합니다.
  2. 내용 입력: '메시지 입력' 칸에 보낼 내용을 적습니다.
  3. 보내기 버튼 누르기: 작성한 메시지 옆의 전송 또는 화살표 아이콘을 누르면 메시지가 보내집니다.

3. 필수 앱 설치하기

스마트폰을 더욱 편리하게 만들어주는 유용한 앱들을 설치하고 사용하는 방법을 알아봅니다.

앱 스토어 열기

  1. 스토어 앱 찾기: 안드로이드폰은 Play 스토어, 아이폰은 App Store 아이콘을 찾아서 누릅니다.

앱 검색 및 설치

  1. 앱 검색: 화면 상단의 검색 창에 설치하고 싶은 앱 이름(예: 카카오톡, 유튜브)을 입력하고 돋보기 아이콘을 누릅니다.
  2. 앱 선택: 검색 결과에서 원하는 앱을 찾아 누릅니다.
  3. 설치 버튼 누르기: 설치 또는 받기 버튼을 누르면 앱이 다운로드되고 자동으로 설치됩니다.
  4. 앱 실행: 설치가 완료되면 열기 버튼을 누르거나, 홈 화면에 새로 생긴 앱 아이콘을 눌러 실행할 수 있습니다.
